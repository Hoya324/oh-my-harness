# Lint & Fix

Run linting and auto-fix for this {{language}} project.

## Commands

1. **Lint check**: `{{linter}} check .`
2. **Auto-fix**: `{{linter}} check . --fix`
3. **Format**: `{{formatter}} .`
4. **Type check**: `mypy .`

## Steps

1. Run the linter to identify issues
2. Apply auto-fix for fixable issues
3. Run formatter for consistent style
4. Run type checker for annotation issues
5. Verify no new issues introduced

## How to use
Run `/lint-fix` before committing to ensure code passes all checks.
