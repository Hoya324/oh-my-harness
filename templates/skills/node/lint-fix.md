# Lint & Fix

Run linting and auto-fix for this {{language}} project.

## Commands

1. **Lint check**: `npx {{linter}} .`
2. **Auto-fix**: `npx {{linter}} . --fix`
3. **Format**: `npx {{formatter}} --write .`

## Steps

1. Run the linter to identify issues
2. Apply auto-fix for fixable issues
3. Manually resolve remaining warnings
4. Run format to ensure consistent style
5. Verify no new issues introduced

## How to use
Run `/lint-fix` before committing to ensure code passes all lint checks.
